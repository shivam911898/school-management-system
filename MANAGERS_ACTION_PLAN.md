## WHAT COULD GO WRONG & HOW WE FIX IT

| Risk                          | How to Prevent                                         |
| ----------------------------- | ------------------------------------------------------ |
| **Teachers won't use it**     | Make grade entry mandatory, provide training & support |
| **Takes longer than planned** | 20% time buffer built-in, clear milestones             |
| **Data gets lost**            | Multiple backups, testing before go-live               |
| **Over budget**               | Fixed-price contracts, quarterly budget review         |
| **Wrong features built**      | Detailed requirements gathering upfront                |
